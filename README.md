# Advent of Code

Solutions to Advent of Code programming puzzles in Scala 3.

## Portfolio

Daily programming puzzle solutions from Advent of Code, implemented in Scala to practice algorithms, problem-solving, and functional programming patterns. Each year's challenges cover various topics including graph traversal, dynamic programming, parsing, and optimization problems.

These solutions showcase functional approaches to algorithmic problems using Scala 3's syntax and features. Used as practice for interview preparation and to strengthen problem-solving skills in a functional paradigm.

## Structure

Solutions organized by year and day, with each puzzle including input parsing, core logic, and both part 1 and part 2 solutions.
